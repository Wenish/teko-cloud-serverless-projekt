### Ausgangslage und Problemstellung

Das Spiel Shadow Infection benötigt für den Vertrieb eine Webseite über die das Spiel Verkauft werden kann. Das Spiel hat 3 Artefakte. Ein Windows Build, ein Linux Build und ein OSX Build. Jedes Build kann mit 500 Megabyte gerechnet werden. Aktuell existiert ein Web Client Prototype. Er stellt dar wie am Ende der Kunde das Spiel kaufen und herunterladen kann. Dieser besitzt noch keine Backend Logik. Für die Zahlungsabwicklung steht Stripe als Anbieter fest. Das Spiel wird als Herunterladbares Spiel verkauft.

Bedeutung Herunterladbares Spiel:
Videospiele im üblichen Sinne, die elektronisch übertragen werden. Sie werden auf ein Gerät mit dauerhaften gewährten Zufriff heruntergeladen. Ausgenommen hiervon sind Spiele, die als Wetten, Glückspiele, Lotterie usw. gelter. (Quelle Stripe)
